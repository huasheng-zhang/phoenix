Core — 核心引擎模块
Agent 主控
core/agent.py
核心对话循环，编排 LLM ↔ Tool 交互。支持流式输出、Skill 注入、记忆加载、上下文窗口管理。
Agent-Tool 循环 流式输出 Skill use/clear 记忆注入 上下文截断 @mention 委派
AgentPool 会话池
core/pool.py
按 (channel, chat_id) 隔离 Agent 实例。懒创建、LRU 淘汰、idle 超时自动清理、共享 MemoryStore。
会话隔离 LRU 淘汰 idle 超时 共享记忆 线程安全
Multi-Agent 协作
core/agents/
Supervisor+Worker 模式，主 Agent 可委派任务给专家 Worker。支持自定义 Agent 创建与切换。
Orchestrator 10 内置角色 delegate_to_agent ask_agent 自定义 Agent Agent 协作讨论
Memory 记忆系统
core/state.py
SQLite-backed 持久记忆。Agent 初始化自动加载、注入 system prompt。跨会话知识共享。
SQLite 存储 key-value 模型 category 分类 自动注入 CLI 管理
Scheduler 调度器
core/scheduler.py
基于 APScheduler 的定时任务调度。单例模式，add/remove 立即生效并持久化到 config.yaml。
Cron 表达式 热加载 自动持久化 Skill 绑定 多渠道推送
Config 配置管理
core/config.py
三层优先级：环境变量 > config.yaml > 代码默认值。支持 ${ENV_VAR} 占位符解析。
YAML 配置 ENV 覆盖 ${VAR} 占位符 validate() .env 加载
