# AI 前沿科技新闻定时推送
# 每天下午 16:35 执行

# 编辑本文件：crontab -e
# 添加以下行：

35 16 * * * /bin/bash /path/to/ai_news_notifier/run_daily.sh >> /var/log/ai_news_cron.log 2>&1

# 说明：
# 35 16 = 每天 16:35
# * * * = 每天、每月、每周都执行
# 第一个 * 是星期（0-7，0 和 7 都是周日）
# 第二个 * 是日期（1-31）
# 第三个 * 是月份（1-12）
# 第四个 * 是星期几（0-6，0 是周日）

# 查看已设置的定时任务：crontab -l
# 删除定时任务：crontab -r
